- `installed_plugins.json` at `~/.claude/plugins/` maps plugin names to install paths; each plugin's skills are in `{installPath}/skills/`
- Skill deduplication: load lowest priority first (plugin), then overwrite with higher (user, project) â€” simple dict key replacement
